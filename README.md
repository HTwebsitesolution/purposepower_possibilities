# Joseph Afolabi — Author Site (Vite + React + Tailwind) — Visual Refresh

Netlify-ready project with upgraded aesthetics inspired by modern author templates.

## Highlights
- Font pairing (Playfair Display + Inter)
- Gradient hero with soft blob backgrounds and 3D tilt book mockup
- Glass cards, gradient buttons, elevated testimonials
- 3 pages + thanks + 404
- SEO meta via `<SEO />`, FAQ JSON-LD on Book page
- Netlify form for the worksheet

## Run
```bash
npm install
npm run dev
```

## Build / Deploy
- Build: `npm run build`
- Publish: `dist/` (Netlify config included)
- Update `public/sitemap.xml` with your domain post-deploy.

## Customise
- Replace images in `public/images/`
- Tweak colours in `tailwind.config.js` and utilities in `src/index.css`
- Edit copy in `src/pages/*.jsx`
